# KIOSC Survey Data Cleaner (Flask Web App)

This project provides a **cloud-hosted web application** designed to simplify and automate the data cleaning process for KIOSC staff. Previously a desktop `.exe` tool, this new version runs on the cloud using **Flask**, making it accessible through any modern web browser without installation.

Survey data collected from various programs often comes in inconsistent CSV formats, which leads to time-consuming and error-prone manual cleaning. This app standardizes that process, ensuring **fast, accurate, and consistent data preparation**.

---

## ✨ Features

- **Web-based Interface**: Accessible via browser; no installation required.  
- **Survey-Specific Cleaning Logic**: Automated cleaning tailored for three distinct survey types (Discovery, VCE, VCES).  
- **CSV Upload & Download**: Upload raw survey CSVs and download cleaned versions instantly.  
- **Error Handling**: Clear feedback for invalid files or incorrect formats.  
- **Scalable Cloud Hosting**: Runs on the cloud, available to all KIOSC staff without local setup.  
- **Maintainable Codebase**: Built with Flask and Python, easy to extend for future survey types or integrations.  

---

## 🛠️ How to Use (for KIOSC Staff)

1. **Access**: Visit the web application link provided by KIOSC (e.g., `https://kioscapps.com.au`).  
2. **Select Survey Type**: Choose which survey dataset you want to process (Discovery, VCE, or VCES).  
3. **Upload CSV**: Use the upload option to select the raw survey CSV file.  
4. **Automatic Cleaning**: The system applies predefined cleaning rules.  
5. **Download Cleaned File**: Once processing is complete, download the ready-to-use CSV file.  
6. **Confirmation**: A success message confirms your data has been processed correctly.
